Definitions

Model_Type: Machine learning model
- MLR = Multiple Linear Regression.
Model Name
- Pillow Impute: Missing data in training period is filled using predictions from correlated snow pillows.
- SnowModel Impute: Missing data in training period is filled using predictions from correlated grid cells from SnowModel simulations (Liston and Elder, 2006).
Seasonality
- season: Model is trained using all ASO flight data.
- accum: Model is trained using a subset of ASO flights that come during accumulation season. The threshold dates to determine accumulation flights were determined using ASO and snow pillow data for each year.
- melt: Model is trained using a subset of ASO flights that come during melt season. The threshold dates to determine melt flights were determined using ASO and snow pillow data for each year.
